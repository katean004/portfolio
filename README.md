# portfolio

Purpose: develop a portfolio using Bootstrap that includes 
1. navbar 
2. a responsive layout (renders well on different devices and screen sizes)
3. responsive images 


Tips:
-Minimizing the use of media queries. 
-Utilize Bootstrap's grid system.
-On xs screen the content should take up the entire screen. 
-On sm and larger screens there should be margin on the sides of the screen. 
-Use HTML validation service to ensure that each page has valid HTML. 


Requirements:
1. navbar consistent on each page
2. navbar on each page contains links to Home/About/Contact and Portfolio pages 
3. all links must work
4. semantic html
5. valid and correct html (validation service)
6. contain personalized information (bio,name,images,links to social media,etc.)
7. properly utilize Bootstrap components and grid system

Bonus:
*Use Bootstrap to make a sticky footer and sub-columns on portfolio site.
